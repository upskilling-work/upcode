# Regras do projeto

## Branches

- Antes de fazer qualquer alteração no código, verifique a branch atual.
- Se a alteração não estiver em uma branch própria (ou seja, estiver na `main`), **pergunte ao usuário se pode criar uma branch** antes de prosseguir. Não altere arquivos na `main` sem confirmação.
- Só após a confirmação, crie a branch e faça as alterações nela.
